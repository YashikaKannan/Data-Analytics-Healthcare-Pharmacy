# Task 1: Basic Healthcare Data Understanding

## 📌 Objective
Work with a healthcare dataset containing patient information such as **Age, Gender, Disease, Medication, and Dosage**.  
The goal is to:
- Explore the dataset
- Identify different types of columns
- Clean missing or duplicate records
- Perform basic analysis (patient count, common diseases, average age)

---

## 🛠 Steps Performed
1. **Load Dataset**  
   - Imported `Pharma_data.csv` using `pandas`.

2. **Inspect Data**  
   - Viewed first rows, column types, and summary statistics.  
   - Checked missing values and duplicates.

3. **Data Cleaning**  
   - Removed duplicate records.  
   - Standardized categorical values (Gender, Disease, Medication).  
   - Filled missing ages with median values.

4. **Basic Analysis**  
   - Total patient count.  
   - Frequency of diseases and medications.  
   - Average age of patients.  
   - Average dosage per medication.  
   - Disease distribution by gender.

5. **Extra Insights**  
   - Correlation between Age and Dosage.  
   - Pivot tables for average age by disease and gender.  
   - Top 3 diseases and medications.

6. **Visualizations**  
   - Disease frequency (bar chart).  
   - Age distribution (histogram).  
   - Average dosage per medication (bar chart).  
   - Age vs Dosage scatterplot.  
   - Correlation heatmap.

7. **Outputs**  
   - Cleaned dataset: `Pharma_data_cleaned.csv`  
   - Summary report: `Pharma_data_summary.txt`  

---

## 📊 Tools & Libraries
- **Python 3.x**
- **pandas** → data handling  
- **matplotlib** & **seaborn** → visualizations  

---

## Results
- Dataset cleaned and standardized.  
- Key insights extracted (most common disease, medication, average age).  
- Visualizations created for better understanding.  
- Ready‑to‑use cleaned dataset and summary report generated for next tasks.
